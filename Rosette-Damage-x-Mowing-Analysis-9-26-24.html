#Analyze Maximum Number of Capitula

Maximum number of capitula is the max number of capitula + buds that were ever present on a plant, after the last mowing treatment happened on (WHICH DATE?) This is because some capitula drop off/blow away as they senesce, meaning that the final assessment of number of capitula may be inaccurate.

#fit model and summarize
fit1 <- glmmTMB(MaxCapitula ~ Treatment + LLLMay2 + (1|Block), data = dat, family = "poisson") #poisson error distribution chosen because this is count data
